# lexiai-spelling-corrector
A two-part NLP system: (1) a domain-specific spelling corrector trained on real AI/NLP arXiv papers using a Bigram LM and Edit Distance, available as a desktop GUI and browser app. (2) a sentiment classifier trained on 50k IMDB reviews with TF‑IDF + Logistic Regression/Naive Bayes, ~89% accuracy, running fully in-browser with no server needed.
