How it works
Text is cleaned and vectorised using TF-IDF (up to 5,000 features, unigrams + bigrams). A Logistic Regression model trained on 40,000 IMDB reviews computes a score. Sigmoid converts this to a probability. Higher = more positive.
Model performance
Trained on 40k IMDB reviews, tested on 10k. Tuned via GridSearchCV (5-fold CV). Accuracy ~89%, ROC-AUC ~0.96. Comparable to published NB-SVM baselines (Wang & Manning, 2012).
